# Example 3

This example reuses the 2024 base story from Example 2, then pairs it with a 2025 story that is only loosely related.  The second story keeps some of the same civic-AI setting, a few recurring people and places, and a similar near-future social atmosphere, but it changes the conflict, scene order, imagery, and narrative purpose.

The case asks how much of the 2025 story was really the 2024 story.  The point of the example is to give `arbd` a low-overlap comparison in which common setting furniture and a handful of reused names should not drive the score very high.
