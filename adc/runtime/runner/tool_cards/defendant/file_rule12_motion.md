File Rule 12 only if one supported ground fits the complaint as pleaded.

- `lack_subject_matter_jurisdiction`: available only in courts that use subject-matter jurisdiction screening.
- `no_standing`: the complaint does not adequately allege injury, traceability, or redressability.
- `not_ripe`: the dispute is too contingent or premature.
- `moot`: the complaint shows no live controversy remains.
- `failure_to_state_a_claim`: the complaint does not adequately plead the claim elements.

Do not use Rule 12 to contest factual allegations that should be tested later.  If the complaint states the claim and the dispute is factual, answer instead.
